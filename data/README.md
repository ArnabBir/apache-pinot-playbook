# Data Files

This directory contains deterministic sample data generated by the repository scripts.

- `merchants.csv` — offline dimension seed used by the batch ingestion example
- `sample_trip_events.jsonl` — append-only fact events
- `sample_trip_state.jsonl` — latest-state updates derived from the events
- `*.kafka.txt` — keyed Kafka-friendly records in `trip_id|json` form

Regenerate them at any time with:

```bash
make generate-data
```
